## Create

* `dotnet new web -o . -n dotnet-minimal-lambda`
* Install libraries for Swagger
    * `dotnet add package Swashbuckle.AspNetCore`
* Add Swagger including UI in `Program.cs` after initial .Build()
```
app.UseSwagger();
app.UseSwaggerUI();
```
